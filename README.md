# attendance-management

管理画面の深いところにあるので、以下の手順をそのまま伝えて操作してもらってください。

-----

### 🛠️ 管理者（情シ）にお願いする操作手順

**対象サイト:
** [LINE WORKS 管理画面（Admin）](https://www.google.com/search?q=https://admin.worksmobile.com/)
※Developer Consoleではなく、通常の「管理画面」からの操作が必要です。

1. **「セキュリティ」** ＞ **「管理者権限」** を開く
2. **「構成員の権限設定」** タブを選択
3. **「追加」** ボタンをクリック
4. **対象者を選択:** あなた（利用者）の名前を検索して選択
5. **権限を指定:** 一覧から **「副管理者」** または **「IT管理」** にチェックを入れる
    * ※特に **「アプリ」** または **「Developer Console」** の管理権限が含まれているロールが必要です。
6. **「保存」** をクリック

### 🔗 あなたがアクセスするURL

権限付与が完了したら、以下のURLからログインできるようになります。
[https://developer.console.lineworks.com/](https://www.google.com/search?q=https://developer.console.lineworks.com/)
